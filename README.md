## Features
- Authentication (JWT or Supabase Auth)
- Course CRUD (Instructor)
- Video lessons (Supabase Storage / S3 / YouTube embed)
- Quiz module and instant scoring
- Per-user progress tracking & certificates
- Optional payment integration (Stripe / Razorpay)
- Admin dashboard with analytics

## Tech Stack
- Frontend: React (Vite/Next.js), Tailwind CSS
- Backend: Node.js, Express
- Database: MongoDB Atlas / Supabase (Postgres)
- Storage: Supabase Storage or AWS S3
- Payments: Stripe / Razorpay
- Deployment: Vercel (frontend) & Render / Railway (backend)
